# MathBlaster — Build Guide

## Option A: Play instantly in browser (no install needed)

Just open this file in Chrome or Edge:
```
game/index.html
```
Works immediately. No server needed for testing.

---

## Option B: Install on Android as PWA (easiest, no Android Studio)

1. Open Chrome on your Android phone
2. Go to the game URL (host it, or use USB tethering from your PC)
3. Tap the **⋮ menu → "Add to Home screen"**
4. The game installs like a native app, plays fullscreen in landscape

To host locally over USB:
```bash
npm install
npm run dev
# Then on phone: visit http://<your-pc-ip>:3000
```

## Option C: Build a real APK with Capacitor

### Prerequisites
| Tool | Download |
|---|---|
| Node.js 18+ | https://nodejs.org |
| Android Studio | https://developer.android.com/studio |
| JDK 17 | included with Android Studio |

Make sure `ANDROID_HOME` is set and `adb` is in your PATH.

### Steps

```bash
# 1. Install dependencies
npm install

# 2. Generate app icons
npm run icons

# 3. Initialize Capacitor (first time only)
npm run cap:init

# 4. Add Android platform (first time only)
npm run cap:add

# 5. Sync web assets into Android project
npm run cap:sync

# 6. Open in Android Studio
npm run cap:open
```

### In Android Studio:
- Wait for Gradle sync to finish
- Connect your phone (enable USB debugging) OR start an emulator
- Click **Run ▶** (Shift+F10)

### Build release APK:
- **Build → Generate Signed Bundle/APK**
- Choose APK → create/use keystore → build
- Output: `android/app/build/outputs/apk/release/app-release.apk`
- Copy to phone and install (enable "Install from unknown sources")

## Game Features

| Feature | Detail |
|---|---|
| Engine | HTML5 Canvas (no dependencies) |
| Enemies | Grunt, Runner, Tank + Boss every 5 levels |
| Shot power | Mega / Power / Normal / Weak (based on answer speed) |
| Tables | 2×–12× scaling with level |
| Levels | Infinite, scaling difficulty |
| Combo | Up to 5× score multiplier |
| Offline | Full offline play via service worker |
| Touch | Large tap targets, touch events on puzzle buttons |
| Screen size | 640×360 base, scales to any screen |
